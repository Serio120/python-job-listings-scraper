# 📄 Fake Jobs Scraper – README

This project contains two Python scripts:

1. **`scraper.py`** – Scrapes job listings from  
   `https://realpython.github.io/fake-jobs/`  
   and saves them into a CSV file.

2. **`view_jobs_html.py`** – Reads the generated CSV file and produces a clean, formatted HTML table for easy viewing.

Both scripts are simple, dependency‑light, and designed for clarity.

---

## 🚀 1. Requirements

### Python version
- Python **3.8+**

### Required libraries
Only one external library is needed:

- **BeautifulSoup4**

You can install it with:

```bash
pip install beautifulsoup4
```

The rest (`requests`, `csv`, `pathlib`) are part of the Python standard library.

---

## 📌 2. Script: `scraper.py`

### What it does
- Downloads the main page of the Fake Jobs website.
- Extracts:
  - Job title  
  - Company name  
  - Location  
  - URL to the job detail page  
- Handles missing fields safely.
- Saves all results into a CSV file named **`jobs.csv`**.

### How to run it

```bash
python scraper.py
```

After running, you should see:

```
Extraction completed. Data saved to jobs.csv
```

A new file `jobs.csv` will appear in the same folder.

---

## 📌 3. Script: `view_jobs_html.py`

### What it does
- Reads the `jobs.csv` file generated by the scraper.
- Builds a clean, styled HTML table.
- Converts job URLs into clickable links.
- Saves the output as **`jobs.html`**.

### How to run it

```bash
python view_jobs_html.py
```

You will see:

```
HTML generated: jobs.html
```

Open the file in your browser to view the job listings in a readable format.
